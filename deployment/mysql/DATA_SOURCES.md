# 演示数据公开来源

本目录中的演示企业数据基于公开官网信息整理，用于离线 SQL 种子数据，不在系统运行时联网抓取。

## 企业来源
- 腾讯官网：[https://www.tencent.com](https://www.tencent.com)
- 字节跳动官网：[https://www.bytedance.com](https://www.bytedance.com)
- 阿里云官网：[https://www.alibabacloud.com](https://www.alibabacloud.com)
- 美团官网：[https://www.meituan.com](https://www.meituan.com)
- 华为官网：[https://www.huawei.com](https://www.huawei.com)

## 整理原则
- 只使用公开可见的企业基础信息，如公司名称、所属行业、官网、典型办公城市与公开业务描述。
- 岗位数据为结合公开招聘方向整理后的**演示化结构化样本**，用于答辩和系统联调，不等同于官网原始职位全文。
- 校友、学生、申请、消息、收藏、账号密码均为本项目演示数据，不对应真实个人。
