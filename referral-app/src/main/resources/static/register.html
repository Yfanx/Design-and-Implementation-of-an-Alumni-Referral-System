Career Profile Setup

先建立一份清晰的求职档案，再进入岗位浏览、申请投递与校友沟通链路。

注册完成后系统会自动登录，并引导你进入学生工作台继续完善资料。 这套入口页与后续页面使用同一视觉语言，答辩展示时更完整。

01 快速建档

先录入账号和基础身份信息，后续在资料页继续完善技能标签与简历附件。

02 岗位匹配

系统会结合目标岗位、城市和技能标签展示更贴近方向的机会。

03 持续跟进

投递后的申请记录、咨询消息与处理状态会在前台持续联动。
