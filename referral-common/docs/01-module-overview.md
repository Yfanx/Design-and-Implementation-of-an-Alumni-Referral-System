# 模块总览

本模块基于 `ruoyi-vue-pro` 的常见分层方式组织，目标是承载“高校校友内推信息管理与对接系统”的核心业务。

核心业务聚合：

1. `AlumniInfo` 校友档案
2. `StudentInfo` 学生档案
3. `CompanyInfo` 企业信息
4. `JobInfo` 岗位信息
5. `ReferralApplication` 内推申请
6. `ConsultMessage` 咨询消息
7. `Dashboard` 数据看板

推荐优先落地顺序：

1. 校友档案维护
2. 岗位发布与审核
3. 学生筛选岗位并提交申请
4. 校友处理申请与咨询
5. 统计看板
